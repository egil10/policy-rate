# Policy Rate

A hyper-minimal dashboard of **central bank policy rates** for ~123 countries and currency unions.
Built for Vercel. Monthly history back to **1945** for the deepest series.

![liquid glass UI](https://flagcdn.com/64x48/eu.png) **Live data sources**

- **Historical (monthly)** — [BIS Central Bank Policy Rates (CBPOL)](https://data.bis.org/topics/CBPOL),
  bulk CSV at `https://data.bis.org/static/bulk/WS_CBPOL_csv_flat.zip`. Monthly, end-of-period.
- **Current snapshot** — [Wikipedia — List of countries by central bank interest rates](https://en.wikipedia.org/wiki/List_of_countries_by_central_bank_interest_rates), snapshotted 2026-05-26.

## What you get

- **123 economies surfaced**: 49 with deep BIS monthly history (24,893 obs), 74 with only the current rate.
- **Multi-line overlay chart** with 5Y / 10Y / 20Y / 50Y / Max time ranges.
- **Filter** by continent, search by name/ISO2 code, **Top 8 highest / lowest**, sort by rate or last change.
- **Flags** from [flagcdn.com](https://flagcdn.com) (1×, 1.5×, 2× retina).
- **Liquid glass** UI — translucent surfaces with backdrop blur, mesh gradient background, system font stack.
- **Lucide** icons throughout.
- Light/dark via `prefers-color-scheme`.

## Run locally

```bash
npm install
npm run data       # downloads BIS zip (~4 MB) and rebuilds public/data/
npm run dev        # http://localhost:3000
```

`npm run data` is split into:

- `npm run fetch` — downloads the BIS CBPOL bulk zip to `data/raw/` and extracts it (needs `unzip`).
- `npm run ingest` — streams the 465 MB CSV and writes per-country files to `public/data/series/`.

The repo ships with pre-built `public/data/` so the dashboard works without running fetch/ingest.


## Coverage

Per `public/data/coverage.json` (regenerated by `npm run ingest`):

- **With deep history** (49 countries / currency unions) — among the longest:
  - **80 years**: Denmark, India, Japan, Sweden, Switzerland, United Kingdom (all 1946 → present, 963–964 monthly obs each)
  - **70+ years**: United States (1954), Greece legacy (1956)
  - **60+ years**: Canada (1960)
  - **50+ years**: Australia (1976), South Africa (1980)
  - Eurozone aggregate (XM) from 1999. Pre-euro national series for Austria, Belgium, France, Germany, Italy, Netherlands, Portugal, Spain run 1945–1998 (labelled "legacy") and then fold into the Eurozone series.
  - Plus all major emerging-market economies: Brazil (1986), Hungary (1987), Russia (1992), Argentina, China, Colombia, Indonesia, Israel, Korea, Mexico, Norway, Philippines, Poland, Saudi Arabia, Thailand, Turkey, and more.
- **Current-only** (74 countries) — current rate + last-change date but no monthly history in this build.
  Examples: Afghanistan, Angola, Bahamas, Belarus, Cape Verde, Eswatini, Kazakhstan, Lebanon, Macau, Mongolia, Samoa, Sint Maarten, Suriname, Uzbekistan, Venezuela, Vietnam, West African States, Zimbabwe, and most smaller African / Pacific / Caribbean economies.

To extend history coverage further, add per-country fetchers in `scripts/` pulling from central-bank pages (e.g. Reserve Bank of Australia historical rates, RBI India, SARB South Africa) and append to `public/data/series/{ISO2}.json`.

## Deploying to Vercel

1. Push to GitHub (this repo is already wired to `origin`).
2. In Vercel: **Add New → Project → Import** the `policy-rate` repo.
3. Framework preset is auto-detected as **Next.js**. Default build command `next build` and output directory work as-is. No environment variables required.
4. Click **Deploy**. The static `public/data/*.json` files are bundled automatically.

Refreshing the data later:

```bash
npm run data
git add public/data
git commit -m "refresh policy rate data"
git push     # Vercel re-deploys
```
